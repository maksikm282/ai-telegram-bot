# AI Telegram Bot / AI Телеграм Бот

Telegram bot that responds to messages using Groq LLM API (Llama 3.3).
Телеграм бот который отвечает на сообщения с помощью Groq LLM API (Llama 3.3).

## Features / Возможности
- Responds to any message using AI / Отвечает на любые сообщения с помощью AI
- Powered by Groq (free, fast) / Работает на Groq (бесплатно, быстро)
- Easy to set up / Простая настройка

## Stack / Стек
- Python
- python-telegram-bot
- Groq API (llama-3.3-70b-versatile)
